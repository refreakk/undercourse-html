
# Чистая сборка для верстальщика

A brief description of what this project does and who it's for


## Старт сборки

Установка пакетов

```bash
npm install
```

Запуск сборки

```bash
npm run build-dev
```

Продакшн сборка

```bash
npm run build-prod
```
## Поддержка

- SCSS, SASS, CSS
- PNG, JPG, SVG, JPEG, GIF, SVG
- Кроссбраузерность
